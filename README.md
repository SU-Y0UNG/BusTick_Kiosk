# 🚌 BusTicket_Kiosk

<p align="center">
버스 승차권 예매를 위한 데스크톱 기반 키오스크 시스템<br/>
C# WinForms 기반으로 직관적인 UI와 효율적인 예매 프로세스를 제공합니다.
</p>

---

## 📌 프로젝트 개요

> 사용자가 오프라인 매표소를 방문하지 않고  
> 목적지 선택 → 좌석 예약 → 결제 → 발권까지  
> 한 번에 처리할 수 있는 키오스크 시스템을 구축합니다.

---

## ⚙️ 주요 기능

### 사용자 인터페이스 (UI/UX) : <span style="color:#6b7280;">WinForms 기반의 직관적인 인터페이스를 제공하며, 노선 검색부터 발권 및 환불까지의 전 과정을 일관된 흐름으로 구성합니다.</span>
### 좌석 관리 시스템: <span style="color:#6b7280;">실시간으로 잔여 좌석을 확인할 수 있도록 구현하고, 중복 예약을 방지하는 로직을 적용합니다.</span>
### 예매 프로세스 자동화: <span style="color:#6b7280;">좌석 정보와 결제 데이터를 연동하여 예약부터 발권까지의 과정을 자동으로 처리합니다.</span>
### 환불 규정 처리: <span style="color:#6b7280;">취소 시점에 따른 수수료를 계산하고, 환불 정보를 표 형태로 시각화하여 제공합니다.</span>
### 외부 서비스 연동: <span style="color:#6b7280;">QR 코드 생성 기능을 제공하며, 모바일 환경과 연동하여 확장성을 확보합니다.</span>

---

## 📈 기대 효과

- **운영 효율성 향상**  
  → 대면 발권 업무를 줄이고 서비스 품질을 향상시킵니다.  

- **사용자 편의성 개선**  
  → 직관적인 UI로 대기 시간을 줄입니다.  

- **데이터 관리 안정성 확보**  
  → 예매 프로세스를 디지털화하여 데이터 정확성을 높입니다.
