# BusTicket_Kiosk
버스 예매를 위한 사용자 친화적인 키오스크 시스템입니다. C# WinForms를 사용하여 구현하였으며, 직관적인 UI와 효율적인 예매 로직을 제공합니다.
🚌 버스 승차권 예매 키오스크 시스템

프로젝트 목표
본 프로젝트는 사용자가 오프라인 매표소 방문 없이 직관적인 UI를 통해 직접 목적지 선택, 좌석 예약, 결제 및 발권을 수행할 수 있는 데스크톱 기반 키오스크 시스템을 구축하는 것을 목표로 합니다.

핵심 기능
사용자 친화적 인터페이스(UI/UX): WinForms를 활용하여 노선 검색부터 발권·환불까지의 전 과정을 직관적인 대시보드 형태로 설계합니다.

좌석 점유 관리: 메모리 내 객체 관리 방식을 통해 실시간 잔여 좌석을 확인하고, 중복 예약 방지 로직을 구현합니다.

프로세스 자동화: 선택된 좌석 정보와 결제 데이터를 매핑하여 예약부터 최종 승차권 발권까지 일련의 과정을 자동 처리합니다.

환불 규정 시각화: 취소 시점에 따른 수수료율을 로직화하고, 이를 표 형태로 시각화하여 명확한 환불 안내를 제공합니다.

외부 서비스 연동: QR 코드 생성 및 앱스토어 연동을 통해 온-오프라인을 잇는 모바일 가이드 기능을 제공합니다.

기대 효과
운영 효율성 증대: 키오스크 도입을 통한 대면 발권 업무 감소로 인력 운영 효율화 및 서비스 품질 향상

사용자 편의성 제고: 직관적인 프로세스와 시각화된 환불 정보를 통해 사용자의 대기 시간 단축 및 서비스 신뢰도 확보

데이터 관리 최적화: 예약 프로세스 전 과정을 디지털화하여 승차권 관리의 데이터 정합성 보장

사용 기술
언어: C#

프레임워크: .NET Framework (WinForms)

기타: GitHub, Sourcetree
