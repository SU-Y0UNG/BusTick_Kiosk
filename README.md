# BusTicket_Kiosk

버스 승차권 예매를 위한 데스크톱 기반 키오스크 시스템입니다.  
C# WinForms를 기반으로 구현하였으며, 직관적인 사용자 인터페이스와 안정적인 예매 프로세스를 제공합니다.

---

## 1. 프로젝트 개요

본 프로젝트는 사용자가 오프라인 매표소를 방문하지 않고  
직접 목적지 선택, 좌석 예약, 결제 및 발권까지 수행할 수 있는  
키오스크 형태의 예매 시스템을 구축하는 것을 목표로 합니다.

---

## 2. 주요 기능

### 2.1 사용자 인터페이스 (UI/UX)
WinForms 기반의 직관적인 인터페이스를 제공합니다.  
노선 검색부터 발권 및 환불까지의 전 과정을 일관된 흐름으로 구성합니다.

### 2.2 좌석 관리 시스템
실시간으로 잔여 좌석을 확인할 수 있도록 구현합니다.  
중복 예약을 방지하기 위한 로직을 적용하고, 메모리 기반 객체 관리 구조를 사용합니다.

### 2.3 예매 프로세스 자동화
좌석 선택 정보와 결제 데이터를 연동하여 예약부터 최종 발권까지의 과정을 자동으로 처리합니다.

### 2.4 환불 규정 처리
취소 시점에 따른 수수료를 계산하는 로직을 구현합니다.  
환불 정보를 표 형태로 시각화하여 사용자에게 명확하게 안내합니다.

### 2.5 외부 서비스 연동
QR 코드 생성 기능을 제공합니다.  
모바일 환경과 연동하여 온·오프라인을 연결할 수 있도록 구현합니다.

---

## 3. 기대 효과

### 3.1 운영 효율성 향상
키오스크를 도입하여 대면 발권 업무를 줄입니다.  
이를 통해 인력 운영 효율을 높이고 서비스 품질을 향상시킵니다.

### 3.2 사용자 편의성 개선
직관적인 사용자 인터페이스를 제공하여 사용 편의성을 높입니다.  
대기 시간을 줄이고 서비스 접근성을 개선합니다.

### 3.3 데이터 관리 안정성 확보
예매 프로세스를 디지털화하여 데이터 관리의 정확성을 높입니다.  
이를 통해 승차권 관리의 안정성을 확보합니다.
