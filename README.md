# 🚌 BusTicket_Kiosk

> 버스 예매를 위한 사용자 친화적인 키오스크 시스템  
> C# WinForms 기반으로 직관적인 UI와 효율적인 예매 프로세스를 제공합니다.

---

## 🎯 프로젝트 목표
사용자가 **오프라인 매표소 방문 없이**  
직접 목적지 선택 → 좌석 예약 → 결제 → 발권까지  
한 번에 처리할 수 있는 **데스크톱 기반 키오스크 시스템** 구축

---

## 🚀 핵심 기능

### 🧑‍💻 사용자 친화적 UI/UX
- WinForms 기반 직관적인 인터페이스
- 노선 검색부터 발권·환불까지 한 화면에서 처리

### 💺 좌석 점유 관리
- 실시간 잔여 좌석 확인
- 중복 예약 방지 로직 구현

### ⚙️ 프로세스 자동화
- 좌석 정보 + 결제 데이터 자동 매핑
- 예약 → 발권까지 전 과정 자동 처리

### 💰 환불 규정 시각화
- 취소 시점별 수수료 계산
- 표 형태로 직관적인 환불 정보 제공

### 🔗 외부 서비스 연동
- QR 코드 생성 기능
- 모바일 연동을 통한 온·오프라인 연결
